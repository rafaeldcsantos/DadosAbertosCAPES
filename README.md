# DadosAbertosCAPES
Alguns exemplos de uso dos dados abertos da CAPES para responder perguntas sobre o SNPG

## GitHub Pages demo

This repository now includes a simple static site structure:

- `index.html` (main page)
- `Demo/` (self-contained app with HTML/CSS/JS/JSON)

### Publish on GitHub Pages

1. Commit and push to GitHub:

```bash
git add .
git commit -m "Add GitHub Pages demo structure"
git push origin main
```

2. In your GitHub repository:
- Go to **Settings** -> **Pages**
- Under **Build and deployment**, choose:
  - **Source**: `Deploy from a branch`
  - **Branch**: `main` and folder `/ (root)`
- Save.

3. Your site will be available at:

- `https://<your-username>.github.io/DadosAbertosCAPES/`
